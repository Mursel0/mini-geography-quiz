# Mini Quiz Game 🌍

A simple geography quiz game built with Python.

## Features
- 4 geography questions
- Multiple choice (a/b/c/d)
- Score tracking
- Correct/Wrong feedback

## How to Run
1. Install Python
2. Run: python mini_quiz.py
3. Answer each question with a, b, c or d

## Topics Covered
- Lists
- Dictionaries
- For loops
- User input
- If/else conditions
